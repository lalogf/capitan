CSS Diner

¡No te preocupes, tu puedes hacerlo!

Estas a punto de aprender sobre selectores CSS (CSS selectors)! Los selectores sirven para definir a qué elementos aplicarle un estilo determinado.

Ejemplo 1 - Una regla CSS

p {
   margin-bottom: 12px;
}

Aqúi, la "p" es el selector (selecciona todos los elementos <p>) y aplica el estilo margin-bottom.

Para jugar escribe el selector CSS en el editor de abajo y selecciona los elementos correctos en la mesa (table en inglés). Si aciertas avanzaras al siguiente nivel.

Pon el mouse sobre los objetos de la mesa para ver las etiquetas HTML que les corresponde.
